Design Web




Design Web, uma das matérias do curso técnico de informática do CEFET-RJ, tem como objetivo ensinar os alunos a fazerem sites com as linguagens de programação propostas.

A matéria é dividida, no geral, em duas partes: a primeira parte é voltada para o aprendizado de HTML e CSS, enquanto a segunda parte é voltada para o aprendizado de JavaScript.

No geral, ambas as partes tem como foco nos preparar para o mercado de trabalho, otimizando nossas habilidades e conhecendo cada vez mais técnicas para melhorar nossos sites, tanto visualmente quanto o programa em si.

Divisão








Como dito anteriormente, a primeira parte da matéria é voltada para o aprendizado de HTML e CSS, onde aprendemos a criar páginas web utilizando as linguagens de marcação e estilo, inicialmente apenas com o básico.

Na segunda parte, aprendemos a utilizar o JavaScript, onde aprendemos a criar páginas web dinâmicas, utilizando a linguagem de programação para criar interações e animações.

Porém, como se pode imaginar, não vemos tudo isso em apenas um ano, como estamos no ensino médio, isso tudo é dividido em 3 anos:

  1. 1º ano:
    • HTML básico
    • CSS básico
  2. 2º ano:
    • HTML avançado
    • CSS avançado
  3. 3º ano:
    • JavaScript básico
    • JavaScript avançado

1º ano: Onde aprendemos a criar páginas web simples, utilizando as tags básicas do HTML e as propriedades básicas do CSS.

2º ano: Onde aprendemos a criar páginas web mais complexas, utilizando as tags avançadas do HTML e as propriedades avançadas do CSS.

3º ano: Onde aprendemos a criar páginas web dinâmicas, utilizando o JavaScript para criar interações e animações.

O que eu acho da matéria


Eu acho a matéria muito interessante, pois é uma área que está em constante evolução e sempre tem algo novo para aprender.

Além disso, é uma área que está em alta no mercado de trabalho, o que é um ponto positivo para nós, alunos.

E, é algo que nos disperta criatividade, pois podemos fazer diversas coisas ao termos a capacidade de fazer um site, principalmente com JavaScript.

Porém, como tudo na vida, tem seus pontos negativos, como o fato de que é uma área que exige muito estudo e dedicação, pois é algo que está sempre mudando e evoluindo.

Enfim, é isso, no geral o processo de fazer um site é cansativo e demorado, além de exigir muito estudo, porém, o resultado é sempre promissor e recompensador :)
